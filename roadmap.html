Development Roadmap Where do we go from here?

The You Are Theme has been neglected since YouAre shut down in 2010, so there is a lot of work to do on the theme. Below is the road ahead.

November 2013

Compatible with WordPress 3.7.1

Currently, the You Are Theme is compatible with 2.8 and it needs to be modernised for compatibility with 3.7.1.

Removing plugin dependency

Right now, the theme is dependent on a plugin and it is my work integrating the archives plugin into the theme so that the original plugin is not required.

Integrate Twitter Bootstrap

To help people customise the theme, it is my intention to tightly integrate Twitter Bootstrap into the You Are Theme.
