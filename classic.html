The Guide

About the below information

Note: These instructions were not written by me, but by the creator of the original theme. Please understand that as the theme is improved, these will be much simpler. Thanks for your understanding. ;)

Installing the YouAre theme is just like installing any other theme. Plugins required and recommended are in the theme that you downloaded.

Recommendation: upload and activate plugins before activating the theme itself. No code/template editing required. Just plugin activation :)

How to install YouAre WordPress Theme

  1. Make sure that you are running WordPress 2.8+ (learn how to upgrade)
  2. Extract the ‘youare.zip’ file
  3. Upload the contents inside the /plugins folder from the zip to your ‘/wp-content/plugins’ directory in your web server (use your FTP client).
    1 plugin is needed (Smart Archives Reloaded) and 5 more are recommended. All plugins are in the youare.zip that you downloaded.
  4. Log in to the WordPress Administration Dashboard. Go to Plugins menu and active plugins uploaded. No code/templates editing required. Just plugin activation :)
  5. Upload the ‘youare’ folder to your ‘/wp-content/themes’ directory in your web server.
  6. Go to WordPress Administration Dashboard and select the ‘Themes’ sub-menu from ‘Appearance’ menu
  7. Select the YouAre theme from the available list of themes to preview
  8. Activate the theme.
  9. Click on the ‘YouAre Options’ sub-menu from ‘Appearance’ menu. Customize. Scroll to the bottom and click “Save”.
  10. To make an archive page: add new page with page template ‘Archives’.
  11. To make a tag cloud page: add new page with page template ‘Tags’.

Plugin required

  1. Smart Archives Reloaded: Go to Plugins menu and activate this plugin before installing the theme. It’s an useful and easy way to display your Archives Page.

Optional plugins

These plugins are very recommended to customize your blog and give it its own personality. Devil is in details:

  1. Better Excerpt: Gives you full control over the WordPress excerpt field.
  2. Top Level Categories: Removes ‘/category’ from the URL for a category (/category/category-name).
  3. Show your Latest YouAre Updates or your Latest Twitter Updates. These plugins are interconnected with theme options.
  4. Contact Form 7: Create a Contact form and paste the code in your ‘Contact Page’.
